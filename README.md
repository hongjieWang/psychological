# psychological
这是一套心理咨询系统，其中包含移动端和服务端。
## 功能简介
### 移动端
- 首页展示
- 咨询师详情页
- 数据下拉刷新，上拉加载
- 登录（手机号验证码、微信登录）
- 微信支付
- 微信分享
- 支付宝支付
- 语音聊天
- 视频聊天
- 客服功能
- 评价功能
- 意见反馈
- 头像上传
- 订单管理
- 我的关注
